1. Install the Django and create a registration page for user.

Fields for registration:

Name
User name
Email
Password (minimum 8 character and should be the combination of Uppercase, lowercase, alphabet and numeric)
Address
Image
There should be validation on client side and server side. Form should submit using Ajax.
2. List of all users, which us registered from registration page.

3. Login page: User can login through user name and password.

4. Dashboard: After successful login, user can see his/her details, which he entered at the time of registration.

5. Logout: User can logout from dashboard
